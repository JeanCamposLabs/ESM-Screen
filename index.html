GitHub token (saved only on this device)

One click pushes the current style, background and toggles to every screen (they follow within ~2 minutes). First time only, it asks for a GitHub fine-grained token (github.com → Settings → Developer settings → Fine-grained tokens). When creating it, two settings are easy to miss: Repository access → Only select repositories → ESM-Screen, and Permissions → Repository permissions → Contents → Read and write. An existing token can be edited and re-used — no need to make a new one.
